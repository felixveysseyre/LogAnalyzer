+++++++++++++++++++++++++++++++++++++++
LogAnalyzer source file processing for deployment instructions
+++++++++++++++++++++++++++++++++++++++

---------------------------------------
1 - Make sure Symfony vendors are installed
---------------------------------------
Look at ./vendors/ directory.
If it is empty, run composer to install the vendors
$ curl -s https://getcomposer.org/installer | php
$ composer.phar install

---------------------------------------
2 - Clear cache and logs
---------------------------------------
$ ./app/console cache:clear
$ ./app/console cache:clear --env=prod

---------------------------------------
3 - Copy your source file into a new directory
---------------------------------------
$ mkdir ../LogAnalyzerToDeploy
$ cp -R ./ ../LogAnalyzerToDeploy

---------------------------------------
4 - Go to the directory
---------------------------------------
$ cd ../LogAnalyzerToDeploy

---------------------------------------
5 - Remove useless files
---------------------------------------
Composer files:
$ rm composer.phar
$ rm composer.json
$ rm composer.lock

GIT files:
$ rm -rf .git
$ rm .gitignore

PHPStorm files (if any):
$ rm -rf .idea

Instructions files:
$ rm DevelopmentInstructions.txt
$ rm ProcessSourcesForDeploymentInstructions.txt

---------------------------------------
5 - Compress the directory
---------------------------------------
$ zip -r ../LogAnalyzerToDeploy.zip ./
